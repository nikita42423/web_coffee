# Задания для команды разработки

## Общая архитектура проекта

**Технологии:** Python, Django, Django Templates, SQLite, Bootstrap 5
**Структура:** Модульная с приложениями в папке `apps/`

## Приложение: Books (Книги)

### Задачи:
1. **Модели (готово):**
   - Модель `Book` с полями: название, автор, описание, текст книги, поля доступности (субтитры, жестовый перевод, аудиоописание), теги, рейтинг

2. **Админ-панель (готово):**
   - Настроить отображение модели Book в админке
   - Добавить фильтры по доступности
   - Добавить поиск по названию и автору

3. **Views (представления):**
   - `book_list()` - список всех книг с пагинацией
   - `book_detail()` - детальная страница книги (готово)
   - `book_search()` - поиск книг по различным критериям
   - `book_by_accessibility()` - фильтрация по типам доступности

4. **Шаблоны:**
   - `books/list.html` - сетка карточек книг с фильтрами
   - `books/detail.html` - детальная страница (готово, требуется адаптация под Bootstrap)
   - `books/partials/book_card.html` - компонент карточки книги

5. **Функциональность:**
   - Добавить возможность оценивать книги (рейтинг 1-5)
   - Добавить комментарии к книгам
   - Реализовать "Избранное" для авторизованных пользователей

## Приложение: Films (Фильмы)

### Задачи:
1. **Модели (готово):**
   - Модель `Film` с полями: название, режиссер, год, описание, продолжительность, поля доступности, теги, рейтинг, ссылка на видео

2. **Админ-панель:**
   - Настроить отображение модели Film в админке
   - Добавить валидацию года выпуска
   - Добавить предпросмотр ссылок на видео

3. **Views:**
   - `film_list()` - список фильмов с фильтрами по году и доступности
   - `film_detail()` - детальная страница фильма
   - `film_by_year()` - фильмы по году выпуска
   - `film_with_subtitles()` - фильмы с субтитрами

4. **Шаблоны:**
   - `films/list.html` - сетка карточек фильмов
   - `films/detail.html` - детальная страница с видео-плеером
   - `films/partials/film_card.html` - компонент карточки фильма

5. **Функциональность:**
   - Интеграция с YouTube/Vimeo (iframe)
   - Система отметок "Просмотрено"
   - Таймкоды для важных моментов фильма

## Приложение: Education (Образование)

### Задачи:
1. **Модели (готово):**
   - Модель `Course` с полями: название, преподаватель, описание, продолжительность, уровень сложности, поля доступности, теги, рейтинг, ссылка на курс

2. **Админ-панель:**
   - Настроить отображение модели Course в админке
   - Добавить выбор уровня сложности через dropdown
   - Добавить валидацию ссылок на курсы

3. **Views:**
   - `course_list()` - список курсов с фильтрацией по уровню
   - `course_detail()` - детальная страница курса
   - `course_by_level()` - курсы по уровню сложности
   - `free_courses()` - бесплатные курсы (если добавить поле price)

4. **Шаблоны:**
   - `education/list.html` - список курсов
   - `education/detail.html` - детальная страница с программой курса
   - `education/partials/course_card.html` - компонент карточки курса

5. **Функциональность:**
   - Прогресс прохождения курса
   - Сертификаты об окончании (шаблоны PDF)
   - Отзывы и рейтинги курсов

## Приложение: Forum (Форум)

### Задачи:
1. **Модели (требуются):**
   - `ForumCategory` - категории форума
   - `ForumTopic` - темы обсуждения
   - `ForumPost` - сообщения в темах
   - `ForumReply` - ответы на сообщения

2. **Админ-панель:**
   - Настроить иерархию категорий
   - Модерация сообщений
   - Статистика активности пользователей

3. **Views:**
   - `forum_index()` - список категорий
   - `category_topics()` - темы в категории
   - `topic_detail()` - обсуждение темы
   - `create_topic()` - создание новой темы
   - `create_post()` - добавление сообщения

4. **Шаблоны:**
   - `forum/index.html` - главная страница форума
   - `forum/category.html` - страница категории
   - `forum/topic.html` - страница обсуждения
   - `forum/forms/` - формы создания тем и сообщений

5. **Функциональность:**
   - WYSIWYG редактор для сообщений
   - Уведомления об ответах
   - Система лайков/дизлайков
   - Возможность прикреплять файлы
